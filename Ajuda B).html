# O Mod necessita de Acesso a internet, 512 MB ram +, processador quadcore 1.2ghz, 5mb de memória livre.

Como Instalar

# Primeiro Baixe a última versão do Mod
# Depois Extraia o Arquivo .zip com seu Extrator de Arquivos. caso não tenha, baixe Aqui o ES File Explorer
# Depois de extraido, ira aparecer outros 2 arquivos além desse o Mod (Caiota.The_Walking_Craft) e a textura.zip "NAO EXTRAIA A TEXTURA, ISSO NAO É NECESSÁRIO".
# Para evitar Travamentos no Block Laucher ao instalar o mod, instale Primeiro a TEXTURA em 🔧/Launcher Options/ Texture options (ON)/Selecionar/textura.zip
# Depois volte ao Menu Inicial do Minecraft Clique na 🔧/ModPE Scripts/selecionar/Caiota.TheWalkingCraft.js.
# Volte, e entre em algum mundo. bom jogo :).
